# drs-server
Open, extensible server implementation of the GA4GH Data Repository Service (DRS) specification

# Installation

Dependencies:
* Java 11+
* Gradle 6.1.1+

```
./gradlew build
```

# Usage

```
./gradlew bootRun
```
